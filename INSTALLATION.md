# Установка и настройка SkyCooker для HomeAssistant

## Быстрая установка

1. **Скачайте репозиторий**
   ```bash
   git clone https://github.com/ClusterM/skykettle-ha.git
   ```

2. **Скопируйте компонент**
   ```bash
   cp -r skykettle-ha/custom_components/skycooker /path/to/your/homeassistant/config/custom_components/
   ```

3. **Перезапустите HomeAssistant**

4. **Добавьте интеграцию**
   - Зайдите в Настройки → Интеграции
   - Нажмите "Добавить интеграцию"
   - Выберите "SkyCooker"

## Настройка устройства

### Для RMC-M40S (и других мультиварок)

1. **Найдите MAC-адрес устройства**
   - Включите Bluetooth на телефоне
   - Найдите устройство "RMC-M40S" в списке доступных устройств
   - Скопируйте MAC-адрес (например: AA:BB:CC:DD:EE:FF)

2. **Настройте интеграцию**
   - **MAC-адрес**: Введите MAC-адрес вашего устройства
   - **Ключ авторизации**: Обычно "000000"
   - **Имя устройства**: Например "Мультиварка Redmond"
   - **Постоянное соединение**: Включите для лучшей производительности

3. **Дождитесь подключения**
   - Интеграция автоматически найдет и подключится к устройству
   - Проверьте статус подключения в карточке устройства

## Возможности после установки

### Для мультиварок

#### Основные элементы управления
- **Мультиварка** - основное устройство с кнопками Start/Stop
- **Режим приготовления** - выпадающий список программ (Рис, Плов, Тушение и т.д.)
- **Температура** - настройка температуры приготовления
- **Время приготовления** - настройка часов и минут
- **Кнопки Start/Stop** - ручное управление процессом

#### Доступные программы для RMC-M40S
1. **Standby** - Ожидание
2. **Multi-chef** - Мультиповар
3. **Rice/Cereals** - Рис/Крупы
4. **Languor** - Томление
5. **Pilaf** - Плов
6. **Frying** - Жарка
7. **Stewing** - Тушение
8. **Pasta** - Макароны
9. **Baking** - Выпечка
10. **Steaming** - На пару
11. **Yogurt** - Йогурт
12. **Dough** - Тесто
13. **Keep Warm** - Поддержание тепла

### Для чайников

#### Основные элементы управления
- **Чайник** - основное устройство
- **Режимы**: Boil (кипячение), Heat (подогрев)
- **Контроль состояния**

## Автоматизация

### Примеры автоматизаций для мультиварки

#### Автоматическое включение утром
```yaml
automation:
  - alias: "Включить мультиварку утром"
    trigger:
      - platform: time
        at: "07:00:00"
    action:
      - service: select.select_option
        target:
          entity_id: select.skycooker_multicooker_mode
        data:
          option: "Rice/Cereals"
      - service: number.set_value
        target:
          entity_id: number.skycooker_multicooker_temperature
        data:
          value: 100
      - service: button.press
        target:
          entity_id: button.skycooker_multicooker_start
```

#### Уведомление о завершении приготовления
```yaml
automation:
  - alias: "Уведомление о завершении приготовления"
    trigger:
      - platform: state
        entity_id: button.skycooker_multicooker
        to: "off"
        from: "on"
    action:
      - service: notify.mobile_app
        data:
          message: "Приготовление завершено! Приятного аппетита!"
```

#### Автоматическое поддержание тепла
```yaml
automation:
  - alias: "Поддержание тепла после приготовления"
    trigger:
      - platform: state
        entity_id: button.skycooker_multicooker
        to: "off"
        from: "on"
    condition:
      - condition: state
        entity_id: select.skycooker_multicooker_mode
        state: ["Rice/Cereals", "Pilaf", "Stewing"]
    action:
      - service: select.select_option
        target:
          entity_id: select.skycooker_multicooker_mode
        data:
          option: "Keep Warm"
      - service: button.press
        target:
          entity_id: button.skycooker_multicooker_start
```

## Устранение неполадок

### Проблемы с подключением

1. **Проверьте Bluetooth**
   - Убедитесь, что Bluetooth включен на сервере HomeAssistant
   - Проверьте, что устройство включено и в режиме сопряжения

2. **Проверьте MAC-адрес**
   - Убедитесь, что MAC-адрес введен правильно
   - Попробуйте перезагрузить устройство

3. **Проверьте ключ авторизации**
   - Попробуйте стандартный ключ "000000"
   - Некоторые устройства могут иметь другой ключ

### Проблемы с управлением

1. **Команды не выполняются**
   - Проверьте статус подключения
   - Попробуйте перезагрузить интеграцию
   - Включите постоянное соединение

2. **Неправильные показания**
   - Проверьте, что устройство поддерживает запрашиваемые данные
   - Попробуйте перезагрузить устройство

## Поддержка

Если у вас возникли проблемы:

1. Проверьте логи HomeAssistant на наличие ошибок
2. Убедитесь, что устройство поддерживается
3. Создайте issue в репозитории с описанием проблемы

## Обновления

Интеграция регулярно обновляется. Следите за обновлениями в репозитории для получения новых функций и исправлений.
