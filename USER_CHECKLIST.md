# Чеклист пользователя: SkyCooker HA для RMC-M40S

## Подготовка

- [ ] **Проверить Bluetooth адаптер** на сервере HomeAssistant
- [ ] **Убедиться, что мультиварка включена** и работает
- [ ] **Найти MAC-адрес мультиварки** (через test_connection.py или Bluetooth сканер)
- [ ] **Подготовить ключ аутентификации** (обычно [0x00, 0x01, 0x02, 0x03, 0x04, 0x05, 0x06, 0x07])

## Установка

- [ ] **Скачать интеграцию** из репозитория
- [ ] **Установить через HACS** или вручную скопировать в custom_components
- [ ] **Перезагрузить HomeAssistant**
- [ ] **Проверить, что интеграция доступна** в списке интеграций

## Настройка устройства

- [ ] **Включить режим пары** на мультиварке (удерживать кнопку питания 3-5 секунд)
- [ ] **Перейти в Настройки → Интеграции → Добавить интеграцию**
- [ ] **Выбрать SkyCooker**
- [ ] **Дождаться сканирования Bluetooth устройств**
- [ ] **Выбрать свою мультиварку** из списка (должна быть RMC-M40S)
- [ ] **Дождаться подключения и аутентификации**
- [ ] **Настроить параметры** (интервал опроса, постоянное подключение)
- [ ] **Сохранить настройки**

## Проверка работы

- [ ] **Проверить появление entity** в HomeAssistant:
  - `switch.skycooker_power` - включение/выключение
  - `select.skycooker_program` - выбор программы
  - `number.skycooker_temperature` - температура
  - `number.skycooker_timer` - таймер
  - `sensor.skycooker_current_temp` - текущая температура
  - `sensor.skycooker_status` - статус

- [ ] **Протестировать основные функции:**
  - [ ] Включение/выключение мультиварки
  - [ ] Выбор программы приготовления
  - [ ] Установка температуры
  - [ ] Установка таймера
  - [ ] Проверка текущей температуры

## Добавление на панель

- [ ] **Перейти в Панели → Редактировать**
- [ ] **Добавить карточку с entity**
- [ ] **Настроить отображение** (выбрать нужные entity)
- [ ] **Сохранить изменения**

## Тестирование программ

- [ ] **Тест программы "Суп":**
  - Выбрать программу "Soup"
  - Установить температуру 90°C
  - Установить время 60 минут
  - Включить мультиварку
  - Проверить, что мультиварка начала работу

- [ ] **Тест других программ:**
  - Проверить работу программ: Porridge, Pilaf, Steam, Bake
  - Убедиться, что все программы доступны и работают

## Автоматизации

- [ ] **Создать автоматизацию** для приготовления:
  ```yaml
  automation:
    - alias: "Приготовить суп вечером"
      trigger:
        - platform: time
          at: "18:00"
      action:
        - service: select.select_option
          target:
            entity_id: select.skycooker_program
          data:
            option: "Soup"
        - service: number.set_value
          target:
            entity_id: number.skycooker_temperature
          data:
            value: 90
        - service: number.set_value
          target:
            entity_id: number.skycooker_timer
          data:
            value: 60
        - service: switch.turn_on
          target:
            entity_id: switch.skycooker_power
  ```

- [ ] **Проверить работу автоматизации**

## Решение проблем

Если возникли проблемы:

- [ ] **Проверить Bluetooth соединение** (test_connection.py)
- [ ] **Убедиться, что мультиварка в режиме пары**
- [ ] **Проверить расстояние до мультиварки** (не более 10 метров)
- [ ] **Перезагрузить Bluetooth адаптер**: `sudo systemctl restart bluetooth`
- [ ] **Проверить логи HomeAssistant** на наличие ошибок
- [ ] **Обратиться к TROUBLESHOOTING.md** за подробными инструкциями
